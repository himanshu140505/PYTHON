# Placement Bootcamp - Interview Preparation

## Structure

```
interview_prep/
├── Day1_Python_Fundamentals/
│   ├── Module1_Fundamentals/        # Python basics, variables, data types, operators, I/O, conditions, loops
│   ├── Module2_DataStructures/      # Strings, lists, tuples, dictionaries, sets
│   ├── Module3_Functions/           # Functions, lambda, recursion
│   ├── Module4_ExceptionHandling/   # Try, except, finally
│   ├── Module5_FileHandling/        # File operations
│   ├── Module6_CodingPractice/      # 20 coding problems to solve
│   ├── Module7_InterviewQuestions/  # 16 interview questions
│   └── Module8_MockInterview/       # Mini mock interview practice
└── README.md
```

## How to Use

1. Start with Module 1 and work through each module sequentially
2. Write your code in the provided Python files
3. Run each file to test your solutions
4. Practice interview questions aloud
5. Time yourself on coding problems (5-10 min each)
6. Review without looking at notes after studying each concept

## Day 1 Schedule

- Module 1: Python Fundamentals (2 hrs)
- Module 2: Data Structures (1.5 hrs)
- Module 3: Functions (included in Module 2 time)
- Module 4-5: Exception & File Handling (included in Module 2 time)
- Module 6: Coding Practice (2 hrs)
- Module 7: Interview Questions (1 hr)
- Module 8: Mock Interview (30 mins)
- Revision (30 mins)

**Total: 6-8 focused hours**

## Rules

- No hints or solutions provided
- If stuck for more than 15 minutes, ask for help
- Speak answers aloud for interview practice
- Focus on understanding, not memorizing
